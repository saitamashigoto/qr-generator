# QRコード生成ツール
## インストール手順
- python, flask, qrcode, npm, nodeを入れてください。
- プロジェクトの中のappディレクトリに入って、以下のコマンドを実行してください。
- `npm install`
- ルートディレクトリに入って以下のコマンドを実行してください。
- `flask run.py`
- ブラウザーでlocalhost:5000を開いてください。
